# FlowPay Frontend

## Installation

```bash
npm install
npm run dev
```

## Backend
Run your Spring Boot backend on:
```bash
http://localhost:8080
```

## Important
Replace:
```javascript
key: 'YOUR_RAZORPAY_KEY'
```

with your actual Razorpay Key ID.

## Features
- Login & Signup
- JWT token storage
- Razorpay payment integration
- Payment verification
- Bootstrap modern UI
